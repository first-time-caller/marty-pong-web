# Pong Game

Welcome to the Pong Game project! This is a simple implementation of the classic Pong game that can be played in a web browser. Below are the details on how to set it up and play.

## Project Structure

The project consists of the following files:

- `index.html`: The main HTML document that structures the webpage and includes references to the CSS and JavaScript files.
- `style.css`: Contains the styles for the webpage, defining the visual appearance of the game elements.
- `script.js`: Implements the game logic, handling user input, updating the game state, and rendering the game.

## How to Play

1. **Open the Game**: Open the `index.html` file in a web browser.
2. **Controls**:
   - Left Paddle: Use the `W` key to move up and the `S` key to move down.
   - Right Paddle: Use the `Up Arrow` key to move up and the `Down Arrow` key to move down.
3. **Objective**: Try to score points by getting the ball past your opponent's paddle. The game continues until a predetermined score is reached.

## Setup Instructions

To set up the game locally:

1. Clone the repository or download the project files.
2. Navigate to the project directory.
3. Open `index.html` in your preferred web browser.
